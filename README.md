# RoDaSoleil Shop - E-commerce сайт с Sanity CMS и Stripe

Современный интернет-магазин одежды с интеграцией Sanity CMS для управления контентом и Stripe для приема платежей.

## Особенности

- ✅ **Sanity CMS** - Удобное управление товарами и категориями
- ✅ **Stripe Payment** - Безопасная обработка платежей
- ✅ **Многоязычность** - Поддержка болгарского, русского и английского языков
- ✅ **Responsive Design** - Адаптивный дизайн для всех устройств
- ✅ **Product Filters** - Фильтрация по категориям, цене, наличию
- ✅ **Wishlist & Cart** - Избранное и корзина с сохранением в localStorage
- ✅ **Image Gallery** - Галерея изображений для каждого товара

## Установка и настройка

1. Установите зависимости: `npm install`
2. Настройте Sanity CMS - см. [SANITY_SETUP.md](./SANITY_SETUP.md)
3. Настройте Stripe - см. [STRIPE_SETUP.md](./STRIPE_SETUP.md)
4. Запустите проект: `npm run dev`
